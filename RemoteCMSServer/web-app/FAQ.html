FAQ - Frequently Asked Questions

  • How do I include remoteCMS managed content on a web page ?

    1. Add the following to the bottom of the page : 
      			<script type="text/javascript">
				var rcmsHost = "http://cms.soa-dna.com/jsFile";
				document.write(unescape("%3Cscript src='" + rcmsHost + "' type='text/javascript'%3E%3C/script%3E"));
			</script>
    2. Wherever you would like a piece of content to appear, insert the following: 
      			<span id="welcome" class="rcms"></span>
      Change the id to something appropriate. Make sure the element has a stylesheet class of rcms. You may include other css classes of your own choosing but it must have a rcms css class. 
      E.g., <span id="welcome" class="someClass rcms"></span>
      You need not use a span tag. You may choose any HTML element that can contain inner HTML within it. E.g. DIV

  • How is content organized on RemoteCMS ?

    One can set up multiple sites specified by a domain name. Each site has multiple pages specified by a path. Each page has multiple content ids. This id maps to the id attribute on the tag you want to replace with your content.
